# FRAM3D

A 3D frame generator for digital assets.

## Project Structure

- `frame-forge/`: The main application code (React/TypeScript/Vite/Three.js).
- `frame-forge-brief.md`: Project brief.
- `frameforge-vs-snapframes.md`: Competitive analysis.

## Development

To start the application:

```bash
cd frame-forge
npm install
npm run dev
```
